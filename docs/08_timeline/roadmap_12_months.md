# 12-Month Roadmap (Draft)

## Month 1–2: Community assessment + governance
- Run CNA surveys (Taipei + Glasgow) for 2–3 weeks
- Conduct 6–10 interviews per location
- Produce CNA synthesis report (Top risks / Top friction / Adoption constraints)
- Finalize safeguarding & data minimization policies
- Define pilot partners and site constraints

## Month 3: MVP delivery (pilot-ready)
- Report intake
- Triage queue + decision log + audit trail
- CAP message composer (text + XML generation)
- Education hub (anti-fraud module v1)
- Basic observability and KPI export

## Month 4–6: Training kit v1 + accessibility
- 3 audience modules + train-the-trainer
- tabletop script + drill script localized per site
- accessibility baseline (large text / screen reader basics)
- baseline KPI collection

## Month 7–9: Dual pilots
- Taipei: 1 pilot site (tabletop + drill)
- Glasgow: 1 pilot site (tabletop + drill)
- Monthly KPI rollups + AARs

## Month 10–12: Scale & handover
- Expand partners (target 4 total)
- Publish Year 1 Impact Report
- Maintenance handbook + governance playbook
- Roadmap v2 + funding strategy
