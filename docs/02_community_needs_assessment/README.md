# Community Needs Assessment (CNA)

This folder contains:
- Taipei (Traditional Chinese): survey + interview guide
- Glasgow/Scotland (English): survey + interview guide

Recommended workflow:
1) Run survey (2–3 weeks)
2) Conduct 6–10 interviews per location
3) Synthesize into “Top risks + Top friction points + Adoption constraints”
4) Convert into MVP scope and KPI baselines
