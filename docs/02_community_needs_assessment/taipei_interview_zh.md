# 台北 CNA 訪談大綱（45–60 分鐘）

## 使用方式
- 先取得口頭同意（可錄音或做筆記）
- 不蒐集敏感個資
- 優先問「最近一次事件」以避免抽象回答

---

## A. 一般受益者（通勤者/學生/社區民眾）
1. 最近一次遇到「可疑/詐騙/公共安全」事件是什麼？你怎麼判斷、怎麼做？
2. 當時你最缺的資訊/協助是什麼？
3. 你最信任哪一種通知來源？（政府/校安/場館/社群/朋友）為什麼？
4. 你最怕什麼：誤報、被打擾、隱私、報復、處理太慢？請排序。
5. 若有「分級通知 + 行動清單」，你覺得哪些內容必須包含？
6. 匿名與資料提供：你願意提供哪些？（時間/地點/照片/聯絡方式）
7. 你願意參加哪種教育活動？（線上/工作坊/tabletop/drill）

## B. 場域管理者（校安/場館/管理室/保全）
1. 目前事件 SOP 是什麼？最大卡點在哪？
2. 你們最怕哪類事件？（人身/群聚/詐騙/假訊息）
3. 你們需要什麼資訊才能判斷真偽？（證據門檻）
4. 對「社區志工協助分流」可行範圍與紅線？
5. 目前通報工具（LINE 群/電話/表單/校務系統）有哪些？哪裡最不順？
6. 你們願意承接的 pilot 類型（tabletop / drill / 小規模部署）？
7. 成效怎麼量測才符合你們 KPI？（處理時間/誤報率/演練通過率）

## C. 合作夥伴（NGO/里辦/扶輪社友）
1. 你們目前做過哪些反詐/安全教育？如何衡量成效？
2. 最缺的是教材、人力、還是工具平台？
3. 你們可提供資源？（場地/講師/志工/宣傳/合作管道）
4. 對外溝通風險（恐慌、政治敏感、媒體）如何控？
5. 你們希望產出哪些交付物（教材包/師培/公開報告/開源工具）？
